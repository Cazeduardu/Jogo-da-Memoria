# 🧠 Jogo da Memória - Django Fullstack

Projeto desenvolvido para a disciplina de [Nome da Disciplina]. O sistema consiste em um jogo da memória interativo com persistência de dados em banco relacional.

## 🚀 Tecnologias Utilizadas
* **Backend:** Django 5.x
* **Banco de Dados:** PostgreSQL
* **Frontend:** JavaScript (ES6+), CSS3 (Animações 3D)
* **Comunicação:** AJAX / Fetch API

## 🛠️ Como rodar o projeto
1. Clone o repositório: `git clone https://github.com/seu-usuario/jogo-da-memoria.git`
2. Crie a venv: `python -m venv venv`
3. Ative a venv e instale as dependências: `pip install -r requirements.txt`
4. Configure suas credenciais do Banco de Dados no `settings.py`.
5. Execute as migrations: `python manage.py migrate`
6. Rode o servidor: `python manage.py runserver`
